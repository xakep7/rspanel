# rspanel
PHP server panel with file manager

# Requirements
> PHP >= 7.1,
> Memcached

PHP extension list:
- PHP-Memcached
- PHP-ssh2 >= 1.2 for auth
- PHP-dom
- PHP-mbstring
- PHP-JSON
- short_tag = on.
